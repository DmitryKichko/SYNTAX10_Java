                                      Преобразование типов в Java:

------------------------------------------------------------------------------------------------------------------------
  Расширение/Неявное/Автоматическое преобразование - от меньшего -> к большему:

  Пример:
  double bigBox = 128;
  System.out.println(bigBox); // на выходе он преобразует и у нас получится дробное число: 1288.0

  Для чего необходимо?

  double res = 10/3; // для того чтобы рассчитать дробные числа

------------------------------------------------------------------------------------------------------------------------
Сужение/Явное/Ручное преобразование - от большего к меньшему (но её не используют):

  int i = (int) 10.89; // (int) - указываем в какой тип данных нужно преобразовать

  System.out.println(i); / на выходе он преобразует и у нас получится целое число: 10
