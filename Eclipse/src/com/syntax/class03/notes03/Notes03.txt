Что необходимо запомнить для условий if, else:

1) if - всегда первый
2) else if - можно использовать несколько раз
3) else - может быть только один и всегда в конце
4) Каждое условие проверяется сверху вниз, пока не найдётся подходящее
5) Как только одно условие сработает - остальные не проверяются
6) Когда оператор if используется без {}, то Java позволяет использовать ТОЛЬКО 1 оператор для каждого блока
то-есть 1 строка для блока if и ТОЛЬКО 1 строка для блока else


В чем разница между = и ==:

1) = - оператор присваивания значения, а не сравнение
2) == - сравнение, выдает true либо false
