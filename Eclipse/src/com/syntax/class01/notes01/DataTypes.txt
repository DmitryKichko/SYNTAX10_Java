

Типы данных - существует 2 типа данных примитивные/непритивные


------------------------------------------ПРИМИТИВНЫЕ ТИПЫ ДАННЫХ - всего их 8------------------------------------------


Числовые типы:

1) byte   // Пример: byte b = 100;
2) short  // Пример: byte s = 10 000;
3) int    // Пример: int i = 1 000 000;
4) long   // Пример: long l = 10 000 000 000L;

Дробные:

5) float  // Пример: float f = 3.14f;
6) double // Пример: double d = 3.14159265359;

Символы:

7) char   // Пример: char c = 'A';

Логические (true/false)

8) boolean // Пример: boolean javaFun = true;

----------------------------НЕПРИМИТИВНЫЕ ТИПЫ ДАННЫХ/ССЫЛОЧНЫЕ - представляют собой объекты----------------------------

НЕПРИМИТИВНЫЕ ТИПЫ ДАННЫХ - представляют собой объекты (Классы/Массивы/Интерфейсы)

1) Классы - любой класс в Java это ссылочный тип

Пример:

class Person {
 String name;
}

Person p = new Person(); // p - это ссылочный тип

Так же:

String s = "Java";
Scanner sc = new Scanner(System.in); // все объекты создаются с помощью ключевого слова new

Все эти переменные содержат ссылку на объект

2) Массивы любого типа (даже примитивного) - это объект, а значит и ссылочный тип.

Пример:

int[] nums = {1, 2, 3}; // nums - ссылка на массив в памяти

3) Интерфейсы - это тип, через который можно ссылаться на объект, реализующий этот интерфейс.

Пример:

Runnable r = new Tread; // Tread - реализует интерфейс Runnable

4) Перечисление (enum) - особый вид классов, тоже ссылочный тип

Пример:

enum Day { MONDAY, TUESDAY }
Day d = Day.MONDAY;

5) Обёртки (Wrapper - классы) - каждому примитиву соответствует объектный класс (обёртка), который тоже является
ссылочным типом

| Примитив | Обёртка |
|   int    | Integer |
|  double  | Double  |
|   char   |Character|
| boolean  | Boolean |
и т.д.

Пример:

Integer x = 10; // Объект, хранящий значение int


Как определить ссылочный тип:

1) Не входит в список 8 примитивов - значит ссылочный
2) Создаётся через ключевоё слово new - почти всегда ссылочный
3) Начинается с заглавной буквы - часто намёк на ссылочный тип
