------------------------------------------------------------------------------------------------------------------------

                                             ИНКРЕМЕНТЫ И ДЕКРЕМЕНТЫ:

------------------------------------------------------------------------------------------------------------------------


Способы для прибавления 1:

    int i = 10;

	1) 	i = i + 1;   // i+1
	2)	i += 1;     // i+1
	3)	i++;       // i+1  - Инкремент

Способы для вычитания 1:

	int a = 10;

	1) 	a = a - 1;   // a-1
    1)  a -= 1;     // a-1
	2)  a--;       // a-1  - Декремент


    10++; ++ или -- не работают с числами, применимы только для ПЕРЕМЕННЫХ
