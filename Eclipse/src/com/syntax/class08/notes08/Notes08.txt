                                                  ЦИКЛЫ В JAVA:

------------------------------------------------------------------------------------------------------------------------


Используется тогда когда мы НЕ ЗНАЕМ сколько раз мы хотим повторить один и тот же блок кода:

- WHILE

Синтаксис цикла WHILE:

while(boolean условие) {
   блок кода который хотим повторить
}

 Как работает шаг за шагом:

 1) Проверяется условие
 2) Выполнение блока кода который хотим повторить

- DO WHILE

Синтаксис цикла DO WHILE:

do {
   блок кода который хотим повторить
} while(boolean условие)

 Как работает шаг за шагом:

 1) Выполнение блока кода который хотим повторить
 2) Проверяется условие


Используется тогда когда мы ЗНАЕМ сколько раз мы хотим повторить один и тот же блок кода:

- FOR

Синтаксис цикла FOR:

 for (инициализация; условие; увеличение/уменьшение) {
      блок кода который хотим повторить
 }

 Как работает шаг за шагом:

 1) Инициализация
 2) Условие
 3) Выполнение блока кода который хотим повторить
 4) Увеличение/Уменьшение


------------------------------------------------------------------------------------------------------------------------

 Ключевое слово break; - используется для выхода из цикла, весь блок кода который идёт после break; будет пропущен

                                                  equals/equalsIgnoreCase

------------------------------------------------------------------------------------------------------------------------

equals - просто используется для сравнения значения строк


equalsIgnoreCase  - используется для сравнения строк и игнорирования регистра (неважно с какой буквы будет написано)
